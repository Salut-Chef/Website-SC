# Salut Chef !

Site web dédiée à la découverte, au partage et à la gestion de recettes culinaires.

---

## Histoire

Etant un cuisinier en reconversion professionnelle, j'ai eu l'occasion de créer l'association **Salut Chef !** pour sensibiliser les étudiants au mieux manger, mieux consommer. Le site venait en tant que soutient complémentaire d'un compte Instagram dédié, me permettant de présenter les recettes tout en rendant le site plus visible.

Depuis Juin 2025, l'association n'existe plus et j'ai dû fermer le site pour des raisons économiques. _En revanche_, je vous laisse le soin de lire le code et regarder ce qui a été réalisé pour ce beau projet !

> Ce projet était mon tout premier gros projet, je me suis amélioré tant sur la qualité du code que sur celle du repo et des commits.  
> Le projet n'est plus en développement, après 2 ans de travail et maintien.

---

## Objectif du projet

**Salut Chef !** est né d’un besoin simple :  
rendre la cuisine accessible à tous les étudiants en proposant des recettes simples, bonnes et pas chères. _Mieux manger et comprendre comment mieux manger !_ Ce site m'a servi également **projet d’apprentissage et d’expérimentation technique** autour du développement web moderne.

Le projet vise à :

- Structurer des fonctionnalités proches d’un produit réel
- Mettre en pratique des concepts front-end et back-end concrets et adaptés (pas de stack overkill)
- Servir de base évolutive pour de futures fonctionnalités (authentification, données dynamiques, etc.)

## Installation locale

### Prérequis

- Node.js ≥ 18
- npm ou yarn

### Installation

```bash
git clone [https://github.com/Salut-Chef/Website-SC.git](https://github.com/HenanAeroo/Website-SC.git)
cd Website-SC
npm install
npm run dev
```
